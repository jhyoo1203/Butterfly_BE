## 사용 기술
- `Java 17.0.12`
- **IDE** : Intellij, VSCode
- **Framework** : Springboot(3.x), Flutter
- **Database** : MySQL(8.0.37), Redis
- **DevOps** : AWS EC2, AWS S3
- **ORM** : Spring Data JPA, QueryDSL
- **API** : KakaoPay API, CoolSMS API, Naver Map API
- **협업 툴** : Notion, Figma, Jira
